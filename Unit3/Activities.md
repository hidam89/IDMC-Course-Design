# UNIT 3

# Classroom Activities

**Course Outcome:** CO3

Configure Secure Agent Groups, Runtime Services, Connections, Asset Management, Scheduling, Performance Optimization, and Monitoring capabilities in Informatica Intelligent Data Management Cloud (IDMC).

---

# Introduction

These classroom and laboratory activities are designed to provide hands-on experience in administering an enterprise IDMC environment. Students will configure Secure Agents, optimize runtime performance, manage assets, automate deployments, and monitor operational performance.

---

# Activity 1

## Secure Agent Group Design

### Objective

Understand the concept of Secure Agent Groups and High Availability.

### Scenario

A multinational company has offices in:

- India
- USA
- Germany
- Australia

Design a Secure Agent deployment strategy.

### Tasks

- Decide the number of Secure Agents.
- Create Secure Agent Groups.
- Explain load balancing.
- Explain failover.

### Deliverable

Architecture diagram with justification.

---

# Activity 2

## Secure Agent Administration

### Objective

Monitor Secure Agent health.

### Tasks

Identify the following:

- Agent Status
- Installed Services
- Runtime Environment

Agent Health Report.

# Activity 3

## Runtime Performance Analysis

### Objective

Understand performance tuning.

### Scenario

A Secure Agent executes 200 mapping jobs daily.

Current problems:

- High CPU usage
- Long queue time
- Slow execution

### Tasks

Recommend:

- JVM Heap Size
- Concurrent Jobs
- Logging Level
- Runtime optimization

### Deliverable

Performance Tuning Report.
