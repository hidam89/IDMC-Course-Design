# Assignment 1

## Secure Agent Group Design

### Objective

Design a Secure Agent deployment architecture for a multinational organization.

### Scenario

A company has offices in:

- India
- USA
- Germany
- Singapore

### Tasks

1. Determine the required number of Secure Agents.
2. Design Secure Agent Groups.
3. Explain High Availability.
4. Explain Load Balancing.
5. Justify your deployment strategy.

### Deliverable

Architecture diagram with a technical report (2–3 pages).
