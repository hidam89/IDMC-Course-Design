# UNIT 3

# Lesson Plan

**Course:** Informatica Intelligent Data Management Cloud (IDMC)

**Unit Title:** Secure Agent Management, Performance Tuning, Asset Management and Monitoring

---

# Course Outcome (CO3)

Configure and administer Secure Agent Groups, Runtime Services, Connections, Asset Management, Scheduling, Performance Tuning, and Monitoring capabilities in Informatica Intelligent Data Management Cloud (IDMC).

---

# Unit Learning Outcomes

After completing this unit, students will be able to:

1. Explain the architecture and benefits of Secure Agent Groups.
2. Configure and manage Secure Agent services and runtime properties.
3. Optimize Secure Agent performance using tuning techniques.
4. Configure enterprise connections and connectors.
5. Manage assets, schedules, and source control.
6. Perform asset import and export using both the UI and CLI.
7. Monitor jobs, events, and operational performance using IDMC monitoring tools.
