# Unit Description

This unit introduces students to Informatica Intelligent Data Management Cloud (IDMC) and its major cloud services. Students will understand enterprise data management concepts, the evolution of Informatica, and the role of Cloud Data Integration (CDI), Cloud Application Integration (CAI), Cloud Data Quality (CDQ), Master Data Management (MDM), Cloud Data Governance and Catalog (CDGC), Cloud Integration Hub (CIH), and Cloud Data Marketplace (CDMP).

---

# Unit Learning Outcomes

After completing this unit, students will be able to:

1. Explain the need for enterprise data management.
2. Describe the evolution of Informatica Cloud.
3. Explain the architecture of Informatica Intelligent Data Management Cloud (IDMC).
4. Differentiate CDI, CAI, CDQ, MDM, CDGC, CIH, and CDMP.
5. Recommend appropriate IDMC services for enterprise business scenarios.
6. Analyze enterprise data integration and data quality challenges.
