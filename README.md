# Overview

This repository contains a comprehensive course design for the Professional Elective course **Informatica Intelligent Data Management Cloud (IDMC)**. The repository is intended to support effective teaching, learning, and assessment by providing structured instructional resources aligned with Outcome-Based Education (OBE) practices.

The course design integrates:

- Course planning
- Lesson planning
- Student-centered learning activities
- Continuous assessment
- Industry case studies
- Active learning pedagogy
- Rubrics
- Teaching resources

# Teaching Philosophy

This course follows:

- Outcome-Based Education
- Active Learning
- Problem-Based Learning
- Case-Based Learning
- Collaborative Learning
- Flipped Classroom

The primary objective is to help students understand enterprise cloud data management by solving realistic business problems rather than memorizing concepts.
