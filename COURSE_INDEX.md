# Course Outcomes

Upon successful completion of this course, students will be able to:

| CO | Description |
|----|-------------|
| CO1 | Explain Informatica Cloud concepts and Enterprise Data Management. |
| CO2 | Configure Informatica Cloud organizations, users, Secure Agents, and connections. |
| CO3 | Apply Secure Agent management, performance tuning, and asset management techniques. |
| CO4 | Configure Secure Agent Groups and troubleshoot Informatica Cloud environments. |
| CO5 | Develop REST API-based automation solutions for Informatica Cloud administration. |
| CO6 | Analyze governance, ethics, and troubleshooting practices in Informatica Cloud. |

Topics Covered:

- Enterprise Data Management
- Evolution of Informatica

- Cloud Application Integration (CAI)
- Cloud Data Quality (CDQ)
- Master Data Management (MDM)
- Cloud Data Governance & Catalog (CDGC)
- Cloud Integration Hub (CIH)
- Cloud Data Marketplace (CDMP)
