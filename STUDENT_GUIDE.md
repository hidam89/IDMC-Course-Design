# Student Guide

## Welcome

Welcome to **Informatica Intelligent Data Management Cloud (IDMC)**.

This course prepares students to understand modern enterprise cloud data management using Informatica Intelligent Data Management Cloud.

---

# Learning Expectations

By the end of this course, students will be able to:

- Explain Informatica Cloud services.
- Configure enterprise cloud environments.
- Analyze enterprise data integration.
- Evaluate cloud governance.
- Develop REST API-based automation concepts.
