# Unit-wise Learning Outcomes

## Unit 1

Students will be able to:

- Explain Informatica Cloud architecture.
- Differentiate CDI, CAI, CDQ, and MDM.
- Describe enterprise cloud services.

---

## Unit 2

Students will be able to:

- Configure organizations.
- Manage users.
- Configure Secure Agents.
- Explain runtime environments.

---

## Unit 3

Students will be able to:

- Configure Secure Agent Groups.
- Create connections.
- Manage assets.
- Monitor enterprise environments.

---

## Unit 4

Students will be able to:

- Explain REST APIs.
- Use REST APIs for administration.
- Analyze troubleshooting techniques.
- Evaluate enterprise automation.
