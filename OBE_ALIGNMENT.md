# Outcome-Based Education (OBE) Alignment

## Course Philosophy

This course is designed following the principles of Outcome-Based Education (OBE), where teaching, learning, and assessment are aligned to achieve measurable Course Outcomes (COs). The course emphasizes student-centered learning, continuous assessment, and real-world applications of Informatica Intelligent Data Management Cloud (IDMC).

---

# OBE Framework

Student Learning Needs
            ↓
Course Outcomes (COs)
            ↓
Teaching & Learning Activities
            ↓
Continuous Assessment
            ↓
Achievement of Outcomes
            ↓
Program Outcomes (POs)

# Graduate Attributes Developed

Students completing this course will develop the ability to:

- Apply enterprise cloud technologies.
- Solve data integration problems.
- Analyze enterprise data quality challenges.
- Design secure cloud-based solutions.
- Communicate technical solutions effectively.
- Work collaboratively in teams.
- Practice ethical and responsible data management.
- Engage in lifelong learning.

The course will be continuously improved based on:

- Informatica product updates

# Expected Learning Outcomes

After completing the course, students should be able to:

- Explain Informatica Cloud architecture.
- Configure enterprise cloud environments.
- Apply Secure Agent and connection management.
- Analyze enterprise case studies.
- Develop REST API-based automation.
- Evaluate governance and ethical practices in enterprise cloud environments.
