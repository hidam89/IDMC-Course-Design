# UNIT 2

# Discussion Questions

**Course Outcome:** CO2

---

# Introduction

The following discussion questions are intended to promote analytical thinking, collaborative learning, and practical application of Informatica Intelligent Cloud Services (IICS) concepts. These questions can be used for classroom discussions, seminars, viva voce, group presentations, or flipped classroom activities.

---

# Section A – Organization and Sub-Organizations

### Q1

Why do multinational organizations require Sub-Organizations instead of managing all users under a single Organization?

---

### Q2

How does Organization Hierarchy improve scalability and governance in enterprise cloud environments?

---

### Q3

Design an Organization Hierarchy for a university with multiple schools and departments. Explain your design decisions.

---

### Q4

What challenges might an organization face if Sub-Organizations are not planned properly?

---

### Q5

How should licenses be allocated among Sub-Organizations to maximize efficiency?

---

# Section B – User Management

### Q6

Why is User Management considered a critical security function in IICS?

---

### Q7

Compare Native Authentication and Salesforce Authentication. Which would you recommend for a multinational enterprise and why?

---

### Q8

What risks arise if inactive user accounts are not reviewed periodically?

---

### Q9

How can organizations improve user lifecycle management?

---

### Q10

Discuss the importance of periodic user access reviews in maintaining cloud security.

---

# Section C – User Groups and User Roles

### Q11

Why are User Groups and User Roles both necessary? Could an organization function effectively using only one of them?

---

### Q12

Explain Role-Based Access Control (RBAC). How does it improve enterprise security?

---

### Q13

Discuss the Principle of Least Privilege. Why is it considered a fundamental security principle?

---

### Q14

When should an organization create a Custom Role instead of using a System-defined Role?

---

### Q15

How do User Groups simplify administration in large enterprises?

---

# Section D – Runtime Environment

### Q16

Why is a Runtime Environment essential in Informatica Cloud?

---

### Q17

---

### Q18

Why is Runtime Environment selection important for enterprise performance?

---

### Q19

How would you design Runtime Environments for Development, Testing, and Production?

---

### Q20

# Section E – Secure Agent

Why is Secure Agent considered the backbone of Hybrid Cloud Integration?

# Section F – Enterprise Case Discussions

### Q26

A hospital uses Oracle Database, SAP ERP, Salesforce, and local file servers.

How would you design:

- Organization hierarchy
- User Groups
- User Roles
- Runtime Environment
- Secure Agent deployment

Justify your design.

---

### Q27

A university wants to migrate from on-premises systems to a hybrid cloud environment.

What administrative challenges are likely to arise, and how would IICS features help address them?

---

### Q28

An organization reports increasing login failures and multiple locked user accounts.

How should the administrator investigate and respond to this situation?

---

### Q29

If your organization expands from one country to ten countries, what changes would you recommend in the IICS administrative structure?

---

### Q30

Imagine you are the Cloud Administrator for a multinational enterprise implementing IICS for the first time.

Discuss:

- Organization structure
- User administration
- Authentication strategy
- User Groups
- User Roles
- Runtime Environment
- Secure Agent deployment
- Security best practices

Present your recommendations as if you were consulting for the organization.

# Suggested Classroom Formats

These questions may be used for:

- Group Discussion
- Think–Pair–Share
- Classroom Debate
- Seminar Presentation
- Viva Preparation
- Case Study Analysis
- Flipped Classroom Activities

---

# Faculty Tips

- Encourage students to justify every answer with enterprise examples.
- Promote discussion rather than memorization.
- Allow multiple valid solutions where appropriate.
- Relate concepts to real-world organizations such as banks, hospitals, universities, and multinational companies.

---

# Expected Learning Outcomes

After participating in these discussions, students should be able to:

- Analyze enterprise administration problems.
- Evaluate different authentication and security strategies.
- Recommend appropriate Runtime Environments.
- Design secure administrative structures.
- Apply IICS concepts to real-world scenarios.
- Communicate technical ideas effectively.
