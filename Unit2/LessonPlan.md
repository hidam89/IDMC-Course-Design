| 5 | User Authentication | Compare Native and Salesforce Authentication | Lecture | Authentication Comparison |
| 6 | User Statistics | Interpret List and Chart Views | Lecture | Dashboard Analysis |
| 7 | User Groups | Configure User Groups | Lecture + Demo | Group Design |
| 8 | User Roles | Explain system-defined and custom roles | Lecture | Role Assignment |
| 9 | Runtime Environments | Explain Runtime Environment Architecture | Lecture | Architecture Drawing |
| 10 | Secure Agent Overview | Explain Secure Agent Architecture | Lecture | Component Mapping |
| 11 | Secure Agent Installation | Install Secure Agent | Demonstration | Installation Walkthrough |
| 12 | Secure Agent Services | Explain Agent Manager and Agent Core | Lecture | Process Analysis |
| 13 | Secure Agent Execution | Run Secure Agent as Local/Network User | Demonstration | Configuration Exercise |
| 14 | Unit Revision | Review and Discussion | Interactive Revision | Quiz + Reflection |
