# Instructor Guide

## Course Overview

This course introduces students to Informatica Intelligent Data Management Cloud (IDMC), covering enterprise cloud data integration, data quality, master data management, governance, Secure Agent administration, runtime environments, REST APIs, and enterprise automation.

The course is designed using Outcome-Based Education (OBE) principles and emphasizes conceptual understanding, critical thinking, and industry applications.

---

# Teaching Philosophy

Students learn best when they actively participate.

The instructor should facilitate learning through:

- Interactive lectures
- Industry examples
- Think–Pair–Share
- Case studies
- Group discussions
- Concept mapping
- Student presentations
- Reflection activities

---

# Classroom Expectations

Students are expected to:

- Attend classes regularly.
- Participate in discussions.
- Complete quizzes and assignments on time.
- Contribute during group activities.
- Present solutions professionally.
- Follow academic integrity guidelines.

---

# Weekly Teaching Workflow

Each session should include:

1. Review previous concepts (5–10 min)
2. Introduce new topic (20 min)
3. Real-world example (10 min)
4. Activity / Discussion (15 min)
5. Summary (5 min)
6. Exit Ticket (5 min)
